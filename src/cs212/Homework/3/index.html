Include Unique fonts in your Website

  1. First go through internet inoder to select a font that you like
  2. Download the font file with suitable extension into your local repository
  3. In this homework i have used a folder called fonts to save all the font files
  4. In the fonts folder i have downloaded a fontfile called CedarvilleCursive-Regular.ttf
  5. Here we use "@font-face" rule which is a CSS rule used to specify a custom font
    for the text which can be loaded form the local or external server
  6. The "@font-face" is used like this: either in the .css file or in the style tag in the head

    @font-face {
    font-family: 'FunkyFontzs';
    src: url('fonts/CedarvilleCursive-Regular.ttf');
    font-weight: normal;
    font-style: normal;
    }

    The url attribute in the font-face rule specifies the location of the font file in the repository
    The font is included to the heading of this website in the following way.

    < h1 style="font-family: 'FunkyFontzs'">< /h1>br
